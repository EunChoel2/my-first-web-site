QR출하 어플리케이션

  • 기능소개
  • 도구
  • React Native를 SDK보다 먼저 손대야 하는 이유

React Native

언어 : Javascript
개발 환경 : Node.js 최신 LTS 버전, Android Studio(iOS 앱 개발은 Mac에서만 가능함)
안드로이드와 IOS 운영체제 동시 사용가능한 크로스 플랫폼
앱의 기본 뼈대와 UI를 설계하는 역할
QR 코드를 스캔한 결과를 화면에 업데이트하거나 카운트를 증가시키는 로직을 처리

SDK (Software Development Kit)

특정 기능을 구현하기 위한 도구 세트
사용할 SDK : react-native-qrcode-scanner (React Native에서 쉽게 QR코드를 스캔할 수 있도록 지원하는 라이브러리)
QR 코드를 스캔할 때 발생하는 이벤트를 처리할 수 있는 기능을 제공
이를 통해 송장 QR 코드와 제품 QR 코드를 스캔하여 카운트를 관리할 수 있습니다.
카메라 기능을 사용해 QR 코드를 인식, 필요한 데이터(송장 번호, 제품 카운트 등)를 처리
